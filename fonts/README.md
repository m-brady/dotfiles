# Custom Fonts

Place custom font files in this directory.

## Installation

The install script will detect fonts in this directory and can help install them.

## Recommended Fonts

The following fonts are installed via Homebrew (see Brewfile):
- Fira Code - A monospaced font with programming ligatures
- JetBrains Mono - A typeface for developers
- Hack Nerd Font - A patched font with icons
- Meslo LG Nerd Font - Another popular patched font

## Manual Installation

To manually install a font on macOS:
1. Double-click the font file
2. Click "Install Font" in the Font Book window
3. The font will be available in all applications

Alternatively, copy font files to `~/Library/Fonts/` for user-specific fonts
or `/Library/Fonts/` for system-wide fonts.
